A simple Python script to rename image files in a folder sequentially.

The script scans a specified directory for image files (`.jpg`, `.jpeg`, `.png`) and renames them in an ordered format such as:
product_001.jpg
product_002.jpg
product_003.png

## Features
- Supports JPG, JPEG, and PNG images
- Sequential numbering with leading zeros
- Prevents overwriting existing files
- Simple command-line usage

## Requirements
- Python 3.x
